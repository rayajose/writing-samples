/ xmen / {id}

Retrieves a specific X-Men entity by its ID.

Parameters

Path

Name Data Type Description
id int The unique identifier for the character.

Query

Name Data Type Description
api_key token The authentication key.

Request

cURL

curl -L http://localhost:8080/api/xmen/{id}?api_key={}

Response

200

# RESPONSE
200 OK
