Required parameters

Required parameters for the xmen PUT/{id} request.

Path

Name Data Type Description
id int The unique identifier for the character.

Header

Name Data Type Description
Content-Type String The media type of the resource sent in the HTTP message body. Use application/json.
API-KEY token The authentication key.

Body

The request body will contain a JSON object with the updated X-Men details.

Name Data Type Description
id int The unique identifier for the character.
name String

The character’s real name.

Format: <<firstName>> <<lastName>>
heroName String The character’s hero name.
power String The character’s powers.
debut String

The comic book title, issue number and year of debut.

Format: <<title>> <<issueNumber>> (<<year>>)
creator String The character’s creators. Use comma to delimit multiple creators.
