/ needles

Parameters

Header

Name Data Type Description
Content-Type string Use application/json
API-KEY token The authentication key.

Body

Name Data Type Description
haystack string The text block to search for word matches.
needles string array The word(s) to match.

Request

This endpoint is better suited for long blocks of text. Returns the match count for each specified word in a block of text. Up to five words can be specified in the request.

cURL

curl -L 'http://localhost:8080/api/needles' \
-H 'Content-Type: application/json' \
-H 'API-KEY: {myApikey}' \
-d '{
    "haystack":"Like finding a needle in a haystack",
    "needles":["finding","needle","a","haystack","needles"]
}'

NodeJs

var request = require('request');
var options = {
    'method': 'POST',
    'url': 'http://localhost:8080/api/needles',
    'headers': {
        'Content-Type': 'application/json',
        'API-KEY': '{myApikey}'
    },
    body: JSON.stringify({
        "haystack": "Like finding a needle in a haystack",
        "needles": [
            "finding",
            "needle",
            "a",
            "haystack",
            "needles"
        ]
    })
};
request(options, function (error, response) {
    if (error) throw new Error(error);
    console.log(response.body);
});

Python

import requests
import json
            
url = "http://localhost:8080/api/needles"
            
payload = json.dumps({
    "haystack": "Like finding a needle in a haystack",
    "needles": [
    "finding",
    "needle",
    "a",
    "haystack",
    "needles"
    ]
})
headers = {
    'Content-Type': 'application/json',
    'API-KEY': '{myApikey}'
}
response = requests.request("POST", url, headers=headers, data=payload)
print(response.text)
