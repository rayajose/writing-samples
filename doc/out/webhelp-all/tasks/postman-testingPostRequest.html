Testing the POST Request

  • Create a Postman workspace.

  • Create a REST API basics collection.

  1. Click the POST Post data request in the collection.
    The POST request type will be pre-selected.
  2. Enter the resource URI in the field next to the request type.
  3. Click the Headers tab.
  4. Populate the Key and Value fields with the API key header information.
