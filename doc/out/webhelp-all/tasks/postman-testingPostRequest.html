Testing POST Request

  • Common Prerequisites:

    • Install Postman desktop application.

    • Sign in to Postman account.

    • Create a Postman workspace.

    • Create a Postman collection using the REST API basics template.

This task describes how to test a POST request in Postman.

To test a POST request follow these steps:

  1. In the xmen collection, select POST Post data request type.

    The POST request builder will be displayed.

  2. Enter the base URL in the URL field: http://localhost:8080/api/xmen
  3. Use the URL field, the Params tab, the Headers tab, and the Body tab, as needed to set the request parameters.
  4. Click Send.
    If successful, the response will show the details for the X-men entity sent in the request.
