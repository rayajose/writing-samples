XmenController

The XmenController class is a RESTful API controller in the marvel.heros package. It provides endpoints for managing X-Men entities, such as creating, updating, retrieving, and deleting X-Men records. This controller is designed to interact with the XmenRepository to perform CRUD operations on the X-Men data.

The class uses Spring Framework annotations to define REST endpoints and inject dependencies. It follows standard RESTful practices with methods for handling HTTP requests such as GET, POST, PUT, and DELETE.

package marvel.heros;               
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.http.ResponseEntity;
import org.springframework.web.bind.annotation.*;               
import java.util.List;
                
@RestController
@RequestMapping("api/xmen")
public class XmenController {
                
    @Autowired
    private XmenRepository xMenRepository;

}

getAllXmen()

@GetMapping //maps method to GET request
public List<Xmen> getAllXmen() {
    return xMenRepository.findAll();
}

@PostMapping //maps method to POST request
public Xmen createXmen(@RequestBody Xmen xMen) {
    return xMenRepository.save(xMen);
}
Description: Creates a new X-Men entity and saves it to the repository.

getXmenById(Short id)

@GetMapping("/{id}") //maps method to GET request
public ResponseEntity<Xmen> getXmenById(@PathVariable Short id) {
    return xMenRepository.findById(id)
    .map(xMen -> ResponseEntity.ok().body(xMen))
    .orElse(ResponseEntity.notFound().build());
}
Description: Retrieves an X-Men entity by its ID.
Parameters: id - Short value
Returns: ResponseEntity<Xmen> - The retrieved X-Men entity or 404 if not found.

updateXmen(Short id, Xmen xMenDetails)

@PutMapping("/{id}") //maps method to PUT request                      
public ResponseEntity<Xmen> updateXmen(@PathVariable Short id, @RequestBody Xmen xMenDetails) {
    return xMenRepository.findById(id)
    .map(xMen -> {
        xMen.setName(xMenDetails.getName());
        xMen.setHeroName(xMenDetails.getHeroName());
        Xmen updatedXmen = xMenRepository.save(xMen);
        return ResponseEntity.ok().body(updatedXmen);
    }).orElse(ResponseEntity.notFound().build());
}
Description: Updates an existing X-Men entity.
Parameters: id - Short value, xMenDetails - ResponseEntity value
Returns: ResponseEntity<Xmen> - The updated X-Men entity or 404 if not found.

deleteXmen(Short id)

@DeleteMapping("/{id}") //maps method to DELETE request
public ResponseEntity<?> deleteXmen(@PathVariable Short id) {
    return xMenRepository.findById(id)
    .map(user -> {
        xMenRepository.delete(user);
        return ResponseEntity.ok().build();
    }).orElse(ResponseEntity.notFound().build());
}
Description: Deletes an X-Men entity by its ID.
Parameters: id - Short value
Returns: 200 OK on successful deletion or 404 if not found.
