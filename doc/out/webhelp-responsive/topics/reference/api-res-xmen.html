REST Resource: xmen

The xmen resource allows you to create, retrieve, update, and delete information about Marvel, X-men characters.

URI

localhost:8080/api/

Parameters

Name Type Data Type Description
API-KEY header token The authentication key.
id body int The unique identifier for the character.
name body string The character’s real name.
heroName body string The character’s hero name.
power body string The character’s power(s).
debut body string The comic book title and year of debut.
creator body string The character’s creator(s).

Properties

Property Data Type Description
id Integer The unique identifier for the character.
name String The character’s real name.
heroName String The character’s hero name.
power String The character’s power(s).
debut String The comic book title and year of debut.
creator String The character’s creator(s).

/ xmen

Create a record in the xmen database.

cURL


            # REQUEST
            curl -L 'http://localhost:8080/api/xmen' \
            -H 'Content-Type: application/json' \
            -X-API-KEY: '{key}' 
            -d '{
            "id": 1,
            "name": "Hisako Ichiki",
            "heroName": "Dust",
            "power": "Sand manipulation",
            "debut": "New X-Men #133 (2002)",
            "creator": "Grant Morrison, Ethan Van Sciver"
            }
