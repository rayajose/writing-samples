Product data specification

Use this specification to format product catalog data for the Shop with Points® shopping site. The Shop with Points® system will import this data for display and order placement on the site. Card members will directly interact with this data to make purchase decisions.

Product data

This is the primary information for product display and order placement on the site.

Field Format Description

id

Required

Format

Max 50 characters

Example

GC001

The product unique product identifier

title

Required

Format

Max 50 characters

Example

Good Clean Skin Care Lotion

The product name

description

Required

Format

Plain Text. Max 5000 characters

Example

Certified organic & water-based 
lotion is pH-balanced
while helping you enhance your 
skin's health. Made with aloe
vera and infused with lemon 
and vanilla. 1.5FL OZ

The product's description

formatted description

Optional

Format

Plain Text. Max 5000 characters

Example

<p>Certified organic & water-based 
lotion is pH-balanced 
while helping you enhance your 
skin's health. Made
with aloe vera and infused with 
lemon and vanilla.</p>
<p>1.5FL OZ</p>

The formatted product description

This is the preferred format if it can be provided in the feed. It will help our product pages mirror partner product pages more closely.

product url

Required

Example

http://www.example.com/products/id

The URL to the website product details page

image url

Required

Example

http:// www.example.com/images/image-main.jpg

The URL to the main product image

additional image url

Optional

Example

http:// www.example.com/images/image-additional-01.jpg

The URL to additional product images

color

Optional

Format

Max 100 alphanumeric characters (max 40 characters per color)

Example

Turquoise

The product color

size

Optional

Format

Max 100 characters

Example

XS

The product size

item group id

Required when variants are intended to roll-up on a single product page.

Optional when variants are intended to display on a separate product pages.

Format

Max 50 alphanumeric characters

Example

AB12345

ID for a group of products that come in different versions (variants)

weight

Required when weight-based shipping will be calculated by the Shop with Points® site.

Format

Number + unit

Example

12 oz

The product weight

Supported units

  • lb

  • oz

  • g

  • kg

Price and availability

These attributes define product price and availability. This information is visible to potential customers.

Field Format Description

availability

Required

Example

in stock

The product availability

Typical values

  • in stock

  • out of stock

  • yes

  • no

availability date

Required when products will be made available for pre-order.

Format

Max 25 alphanumeric characters

Example

YYYY-MM-DDThh:mm [+hhmm]

The product availability date (when the product can ship)

price

Required

Format

Numeric

Example

27.00

The product price

sale price

Required

Format

Numeric

Example

19.00

The product sale price

minimum price

Required if limiting sale price suggestions to a minimum price e.g. to comply with local pricing laws or to indicate a MAP (minimum advertised price).

Example

19.00

The product minimum price

Product identifiers

These attributes define the products you're selling in the global marketplace.

Field Format Description

brand

Required

Format

Max 70 characters

Example

Good Clean

The product brand name

gtin

Required

Format

Max 50 numeric characters

Example

3234567890126

The product Global Trade Item Number (GTIN)

Supported formats

  • UPC (in North America / GTIN-12) 12-digit number like 323456789012, 8-digit UPC-E codes should be converted to 12-digit codes

  • EAN (in Europe / GTIN-13) 13-digit number like 3001234567892

mpn

Required if your product does not have a manufacturer assigned GTIN

Format

Plain Text. Max 5000 characters

Example

ACME010000123

The product Manufacturer Part Number (MPN)
