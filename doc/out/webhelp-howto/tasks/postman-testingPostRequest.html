Testing POST Request

  • Prerequisites:

    • Install Postman desktop application.

    • Sign in to Postman account.

    • Create a Postman workspace.

    • Create a Postman collection using the REST API basics template.

This task describes how to test a POST request in Postman.

To test a POST request follow these steps:

  1. In the xmen collection, select POST Post data request type.

    The POST request builder will be displayed.

  2. Enter the following in the URL field.
    http://localhost:8080/api/xmen
  3. Click Headers and populate the Key and Value fields.
    For more information, see xmen POST Parameters
  4. Click Body and enter the parameters to create a new X-men character.
    For more information, see xmen POST Parameters
  5. Click Send.
    A successful response will show the details for the X-men character sent in the request body.

    For more information, see xmen POST Response
